= Setup Your Development Environment

In this workshop, you will use GitHub Codespaces as your development environment and a Neo4j Sandbox as your database.

== Launch GitHub Codespaces

Open the workshop repository in GitHub Codespaces:

link:https://codespaces.new/neo4j-graphacademy/genai-maf-context-providers[image:https://github.com/codespaces/badge.svg[Open in GitHub Codespaces]^]

When the Codespace starts, it will automatically:

* Install Python 3.12 and all required packages (including `agent-framework`, `openai`, `neo4j-graphrag`, and Jupyter)
* Open the setup instructions and environment configuration files

== Create a Neo4j Sandbox

You will need a Neo4j instance with the **recommendations** dataset — a movie database with actors, directors, genres, and user ratings. The sandbox includes vector indexes on movie plot embeddings that you will use for semantic search in later labs.

. Go to link:https://sandbox.neo4j.com[sandbox.neo4j.com^]
. Sign up for a free account or sign in if you already have one
. On the next screen, you will see **Select a project**. Scroll down and choose the **Recommendations** dataset:
+
image::recommendations.png[Select the Recommendations dataset]
+
. Click **Create** to launch your sandbox
. Once created, download your credentials and save them somewhere safe — you will need the **Bolt URL**, **username**, and **password** in the next step

IMPORTANT: Save your sandbox credentials now. You will not be able to retrieve the password later.

== Configure Environment Variables

In your Codespace, create a `.env` file by copying the example:

[source,bash]
----
cp .env.example .env
----

Open the `.env` file and update it with your credentials:

[source]
----
OPENAI_API_KEY=sk-your-key-here
OPENAI_RESPONSES_MODEL_ID=gpt-4o
NEO4J_URI=neo4j+s://your-sandbox-id.databases.neo4j.io
NEO4J_USERNAME=neo4j
NEO4J_PASSWORD=your-sandbox-password
----

For the **OpenAI API key**, you can either:

* Use your own key from link:https://platform.openai.com[platform.openai.com^]
* Use the credentials provided to you by the workshop instructor

== Verify Your Setup

Run the test environment script to verify everything is configured correctly:

[source,bash]
----
python genai-maf-context-providers/test_environment.py
----

You should see confirmation that your OpenAI and Neo4j connections are working.
