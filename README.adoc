= Self-Grounding Agents with MAF and Neo4j

A workshop site for the Self-Grounding Agents with MAF and Neo4j course, built with https://antora.org[Antora].

== Prerequisites

* https://docs.astral.sh/uv/[uv] (Python script runner)
* https://nodejs.org[Node.js] 20+

== Setup

Install the Antora dependencies:

[source,bash]
----
cd site
npm install
----

Create a `.env` file in the repository root with the path to the course content:

[source]
----
COURSE_DIRECTORY=/path/to/courses/asciidoc/courses/genai-maf-context-providers
----

== Syncing Course Content

The `sync_course.py` script converts GraphAcademy course content into the Antora site structure. Run it whenever the course content changes:

[source,bash]
----
uv run sync_course.py
----

This will:

* Read the course from `COURSE_DIRECTORY`
* Convert lessons into Antora pages under `site/modules/ROOT/pages/`
* Generate the navigation file (`site/nav.adoc`)
* Copy images to `site/modules/ROOT/images/`

== Building the Site

[source,bash]
----
cd site
npm run build
----

Output is generated in `site/build/site/`.

== Local Preview

[source,bash]
----
cd site
npm run serve
----

Opens http://localhost:8080 in your browser.

== Deployment

Pushing to `main` triggers the GitHub Actions workflow (`.github/workflows/deploy-antora.yml`) which builds and deploys to GitHub Pages.
